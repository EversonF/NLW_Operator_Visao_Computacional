# 👁️ Vision Guard

![Python](https://img.shields.io/badge/Python-3.10-blue)
![OpenCV](https://img.shields.io/badge/OpenCV-Computer%20Vision-green)
![PyTorch](https://img.shields.io/badge/PyTorch-AI-red)

Projeto desenvolvido durante o **NLW Operator** da Rocketseat na trilha de **Visão Computacional com Python**.

O objetivo da aplicação é utilizar **Inteligência Artificial para detectar objetos em tempo real através da webcam**, aplicando técnicas de **Computer Vision e Machine Learning**.

---

## 🚀 Tecnologias utilizadas

- Python
- OpenCV
- PyTorch
- Hugging Face
- Streamlit

---

## 🧠 Conceitos aplicados

Este projeto utiliza conceitos de:

- Computer Vision
- Processamento de imagens
- Detecção de objetos
- Inferência de modelos de Machine Learning
- Captura e processamento de vídeo em tempo real

---

## 📷 Funcionamento da aplicação

O sistema realiza o seguinte fluxo:

1. Captura vídeo em tempo real pela webcam
2. Processa cada frame da imagem
3. Aplica um modelo de IA para detectar objetos
4. Desenha caixas delimitadoras nos objetos identificados
5. Exibe o resultado em uma interface interativa

---

## 🏗️ Arquitetura simplificada

Webcam <br>
↓ <br>
OpenCV (captura de imagem) <br>
↓ <br>
Modelo de IA (PyTorch / HuggingFace) <br>
↓ <br>
Detecção de objetos <br>
↓ <br>
Interface com Streamlit <br>

## 📦 Instalação

Clone o repositório:

``bash
git clone https://github.com/seu-usuario/vision-guard.git

Acesse a pasta do projeto:
cd vision-guard

Crie um ambiente virtual:
python -m venv venv

Ative o ambiente virtual:

Windows:
venv\Scripts\activate

Linux / Mac:
source venv/bin/activate

Instale as dependências:
pip install -r requirements.txt``

---

## ▶️ Executando o projeto

Para iniciar a aplicação:
streamlit run app.py

Após iniciar, o sistema abrirá automaticamente no navegador.
